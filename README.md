## Project Summary
This is a personal portfolio website designed and developed to showcase the skills, projects, and services of Malik M Abdullah. It includes sections for **Home**, **About Me**, **Services**, **Projects**, and **Contact**, along with interactive elements such as a typing animation, hover effects, and a project slider. The website is fully responsive and designed to provide a modern, user-friendly experience across all devices.

## Color Palette and Typography

### **Colors**
| Name | Code | Usage |
|------|------|-------|
| Primary Black | #2d2d2e | Background |
| Accent Red | #e50914 | Buttons, headings, highlights |
| Light Grey | #f2f2f2 | Text |
| Card Background | #333333 | Service and project cards |

- **Headings:** Montserrat, sans-serif
